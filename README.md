# dirtree

#### 介绍
当你向AI打听一些知识，并且需要让AI知道你的目录结构时，请使用此包生成目录信息并传达给AI。
我的名字是立方体,我会开发更多简单易用的工具

#### 软件架构
>python3.8

#### 安装教程
1.  pip install dirtree

#### 使用说明
1.  dirtree -p "/xx/xx"
